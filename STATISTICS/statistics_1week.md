# 1️⃣ 개념 정리 
## 01. 데이터를 분석하다

```
✅ 학습 목표 :
* 데이터에 대해 이해한다.
* 데이터 분석에 대해 이해한다.
```
데이터는 수치의 모음이지만 그 자체로는 무엇인지 알 수 없기 때문에 데이터를 적절하게 분석해야하며, 이를 통해 대상을 이해하거나 미래를 예측할 수 있습니다.

데이터 분석의 목적은 총 3가지로 데이터 요약, 대상 설명, 새로 얻을 데이터를 예측하는 것으로 나눌 수 있습니다.
통계학에 의거한 데이터 분석은 데이터를 정량적이고 객관적으로 평가하여 대상이 가진 성질과 관계성을 올바르게 찾고자 하는 시도입니다.


## 02. 통계학의 역할

```
✅ 학습 목표 :
* 산포에 대해 설명할 수 있다.
```

산포: 데이터에 포함된 값 하나하나의 차이
데이터 퍼짐은 대상이 가진 성질이나 관계성의 본모습을 감추고 정확하게 파악할 수 없도록 한다.


## 03. 통계학의 전체 모습
```
✅ 학습 목표 :
* 기술통계와 추론통계의 차이점을 설명할 수 있다.
* 통계적 추론과 가설 검정에 대해 이해한다.
```
1. 기술통계는 수집한 데이터를 정리하고 요약하는 방법을 의미하며 데이터 자체의 성질을 이해하는 것을 목표로한다.
추론통계는 수집한 데이터로부터 데이터의 발생원을 추정하는 방법을 의미한다.

2. 통계적 추론은 데이터에서 가정한 확률 모형의 성질을 추정하는 방법이다.
가설검정은 세운 가설과 데이터가 얼마나 들어맞는지를 평가하여 가설을 채택할 것인가를 판단하는 방법이다.


## 04. 데이터 분석의 목적과 알고자 하는 대상

```
✅ 학습 목표 :
* 데이터 분석의 목적에 맞게 ‘알고자 하는 대상(모집단)’을 명확히 정의하는 것이 왜 중요한지 이해한다.
```

알고자하는 대상의 규모에 따라 모든 데이터를 얻을 수 있는지 없는지가 결정된다. 예를들어 신약 효과를 검증하고싶을때는 실험참가자 80명이 아니라 참가하지 않은 수많은 고혈압 환자가 대상이다. 하지만 고등학교 2개 반 중 평균 시험 점수가 높은 반이 승리한다고 했을때 필요한 데이터는 80명의 데이터가 전부이다.


## 05. 모집단

```
✅ 학습 목표 :
* 모집단의 개념을 명확히 설명할 수 있다.
```

모집단: 알고자 하는 대상 전체
'지금 알고자 하는 대상은 무엇인지' 그리고 '무엇을 모집단으로 설정할 것인지'의 문제는 항상 주의를 기울여야 한다.
모집단 크기: 모집단에 포함된 요소의 수
    - 유한모집단: 모집단 중 한정된 요소만 포함. ex) 2개의 반 80명, 한국인
    - 무한모집단: 모집단 중 요소의 개수가 무한한 것 ex) 주사위, 신약 효과 예시


## 06. 모집단의 성질을 알다 

```
✅ 학습 목표 :
* 전수조사와 표본조사의 개념적 차이와 각각의 필요성을 설명할 수 있다.
* 표본에 대해 이해한다.
```

1. 전수조사: 모집단에 포함된 모든 요소 조사, 모집단이 유한모집단일 때 선택
그러나 실현 불가능한 경우가 많음

2. 표본조사: 표본을 이용해 모집단의 성질을 조사하는 것 ex) 선거 출구조사
(추론통계: 모집단의 일부를 분석하여 모집단 전체의 성질을 추정)
(표본: 추론통계에서 조사하는 모집단의 일부)

3. 표본크기: 표본에 포함된 요소의 개수
모집단의 성질을 추정할 때의 확실성이나 가설검정의 결과에도 영향을 끼치므로 통계분석에 있어 중요한 요소 중 하나임

# 2️⃣ 확인 문제

## 문제 1.

> **🧚Q. 한 회사가 직원들의 ‘업무 만족도’를 파악하기 위해 설문조사를 실시했다.
전체 직원 1,000명 중 50명에게만 설문을 진행했고, 그 결과 평균 만족도는 4.5점(5점 만점)이었다.
회사는 이를 근거로 “우리 회사 직원들은 대체로 업무 만족도가 높다”고 결론 내리려 한다.**
>
> **이 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 무엇인가?**

~~~
1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
2️⃣ 만족도의 평균값만으로 결론을 내려도 되는가?
3️⃣ 설문 문항이 객관적으로 구성되었는가?
4️⃣ 이 결과를 모집단(전체 직원)으로 일반화해도 되는가?
~~~



<!--학습한 개념을 활용하여 자유롭게 설명해 보세요.-->

```
답: 1️⃣
조사에서 가장 먼저 선행되어야하는 것은 '무엇을 모집단으로 설정할 것인지'에 대한 문제이다.
우리는 조사한 50명의 업무 만족도를 파악하는 것이 아니라 회사 전체 직원 1000명의 업무 만족도를 파악하는 것이 목적이다. 모집단은 회사 전체 인원 1000명이 되며 이중 50명의 표본을 뽑아 표본을 통해 모집단의 성질을 조사하고 있다. 따라서 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 조사 대상 50명이 전체 직원을 대표하는 표본이 맞는지가 될 것이다.
```
