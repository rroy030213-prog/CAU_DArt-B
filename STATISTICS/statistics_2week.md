# 📊 데이터 분석을 위한 기초 통계학 정리

## 1. 기초 통계 용어 (Basic Terminology)

### 🔹 변수 (Variable)
* **정의:** 데이터 중 공통의 측정 방법으로 얻은 같은 성질의 값입니다.
* **차원 (Dimension):** 변수의 개수를 표현하는 단위입니다.
    * **고차원 데이터:** 여러 개의 변수를 포함하고 있는 복잡한 데이터를 의미합니다.

---

## 2. 데이터의 유형 (Data Types)
데이터 분석의 시작은 내가 가진 변수가 어떤 유형인지 파악하는 것입니다. 크게 **양적 변수**와 **질적 변수**로 나뉩니다.

### 🟢 양적 변수 (Quantitative Variable)
> 숫자로 나타낼 수 있으며, 대소 관계가 있고 사칙연산(평균 등)이 가능한 변수

1.  **이산형 (Discrete):**
    * 값이 끊어져서 존재하며 셀 수 있는 데이터
    * *ex) 주사위 눈, 사람 수, 사고 건수*
2.  **연속형 (Continuous):**
    * 간격 없이 이어지는 실숫값으로 나타낼 수 있는 데이터
    * *ex) 키, 몸무게, 온도, 시간*

### 🔵 질적 변수 (Qualitative Variable)
> '범주형 변수'라고도 하며, 숫자가 아닌 특성이나 범주로 분류되는 변수

* 수치적 연산이 불가능하며 상태나 종류를 나타냅니다.
* *ex) 혈액형, 성별, 메뉴 이름, 동전의 앞/뒤*

---

### 📌 요약 표 (Cheat Sheet)

| 구분 | 종류 | 특징 | 대표 예시 |
| :--- | :--- | :--- | :--- |
| **양적 변수** | 이산형 | 셀 수 있는 정수 형태 | 인원수, 자녀 수 |
| | 연속형 | 소수점으로 이어지는 값 | 키($cm$), 무게($kg$) |
| **질적 변수** | 범주형 | 종류를 나타내는 이름 | MBTI, 거주 지역 |


## 02. 데이터 분포

```
✅ 학습 목표 :
* 변수 유형에 따라 히스토그램 정의를 설명할 수 있다.
```

변수 유형에 따라 히스토그램 정의
1. 이산형 양적 변수의 히스토그램
- 가로축은 숫자, 세로축은 데이터에 나타난 개수(도수, 빈도, 횟수)를 표시

2. 연속형 양적 변수의 히스토그램
- 범위를 설정하고 그 범위에 포함되는 숫자 개수를 세어 세로축에 둠. 이 범위의 넓이를 '구간폭'이라고 부름
- 같은 데이터라도 구간폭에 따라 주는 인상이 다르다.

3. 범주형 변수의 히스토그램
- 가로축에 각 범주, 세로축에 각 범주에 속하는 개수 나타냄
- 가로축 순서에 특별한 뜻은 없음

히스토그램은 빈도를 대략 파악할 수 있고 이상값 확인을 위해서도 중요하지만 주관적 판단에 일임되기 때문에 데이터를 정확히 기술하거나 대상을 이해하려는 목적이 달성되지 않음
따라서 수치적으로 다루는 통계분석 필요

## 03. 통계량
```
✅ 학습 목표 :
* 통계량의 개념과 역할을 이해한다.
* 대푯값의 종류와 각 대푯값의 특성을 설명할 수 있다.
* 자료의 산포를 나타내는 분산과 표준편차의 의미를 이해한다.
* 이상값(이상치)의 개념과 통계 분석에서의 영향을 이해한다.
```
- 통계량의 개념과 역할
통계량: 수집한 데이터로 이런저런 계산을 수행하며 얻은 값
데이터 그 자체의 성질을 기술하고 요약하는 통계량을 기술통계량 or 요약통계량이라고 부름 ex) 평균값, 표준편차 등
기술통계량은 주로 양적 변수를 대상으로 계산한다.

- 대푯값의 종류, 각 대푯값의 특성
대푯값:대략적인 분포 위치, 즉 대표적인 값을 정량화하기 위해 사용하는 통계량
1. 평균값: '표본평균'이라고도 함. 평균값 공식
2. 중앙값: 크기 순으로 값을 정렬했을 때 한가운데 위치한 값
극단적으로 크거나 작은 값이 있어도 영향을 받지 않음
3. 최빈값: 데이터 중 가장 자주 나타나는 값
어떤 값이 전형적으로 나타나는가를 파악할 때 도움이 됨

대푯값은 n개인 데이터를 대푯값으로 집약함으로써 분포의 위치를 정량적으로 특정 지을 수 있으나 데이터의 분포나 최댓값, 최솟값 등 몇 가지 정보는 읽을 수 없기 때문에 다른 통계량도 함께 확인하는 것이 중요하다.
처음부터 히스토그램을 그려 대략적인 파악을 한 다음 대푯값으로 분포를 특징 지을 수 있는지 확인하는 것이 중요한 데이터 분석 작업의 순서이다.

- 자료의 산포를 나타내는 분산과 표준편차의 의미
(표본)분산: 표본의 각 값과 표본평균이 어느 정도 떨어져 있는지를 평가하는 것으로 데이터 퍼짐 상태를 정량화한 통계량
<분산 공식>
데이터 퍼짐 정도가 크면 s^2이 커짐

(표본)표준편차s: 표본분상의 제곱근을 취한 값
<표준편차 공식>
분산과 표준편차에는 포함하는 정보에 차이가 없음

- 이상값(이상치)의 개념과 통계 분석에서의 영향 이해
이상값: 평균값에서 표준편차의 2배 또는 3배 이상 벗어난 숫자
- 시각화를 통해 이상값이 있는지 확인하고 실수인 경우 데이터에서 제외하여 이후 분석에 사용되지 않도록 해야함.

## 04. 확률

```
✅ 학습 목표 :
* 확률의 기본 개념과 확률변수 및 확률분포의 의미를 이해한다.
* 추론통계에서 사용되는 확률분포의 개념을 이해한다.

```

확률의 기본 개념과 확률변수, 확률 분포 의미 이해
확률: 불확실한 사건의 발생 가능성을 숫자로 표현한 것, P(A)로 나타냄
확률변수: 확률이 달라지는 변수, 확률변수가 실제로 취하는 값을 실현값이라고 한다.
- 이산형 확률변수와 연속형 확률변수로 나뉜다.
확률분포: 가로축에 확률변수, 세로축에 그 확률변수의 발생 가능성을 표시한 분포
- 연속형 확률변수의 경우, 값에 일정한 범위를 두고 확률을 구하는데 이 확률을 계산하는 함수를 확률밀도함수라고 한다.

추론통계에서 사용되는 확률분포의 개념 이해
추론통계는 모집단의 일부인 표본에서 모집단의 성질을 추정하고자 한다.
현실 세계의 모집단을 수학 세계의 확률분포로 가정하고, 표본 데이터는 그 확률분포에서 생성된 실현값인 것으로 가정하여 분석 진행 -> 모집단과 표본 데이터처럼 다루기 어려운 대상이 확률분포와 그 실현값처럼 다룰 수 있는 대상으로 치환되는 것이다.


사건 간 독립의 개념과 통계적 의미 이해
독립: 한쪽이 어떤 값을 취하든지, 다른 한쪽의 발생 확률은 변하지 않는다는 것을 의미한다.
X와 Y의 동시확률분포P(X,Y)는 각각의 확률 P(X)*P(Y)와 같다.


## 05. 이론적인 확률분포

```
✅ 학습 목표 :
* 모수의 개념과 통계적 의미를 이해한다.
* 정규분포의 특성과 표준화의 개념을 설명할 수 있다.
```

모수의 개념과 통계적 의미 이해
모수(파라미터): 이론적인 확률분포는 수식으로 표현된다. 이때 분포의 형태를 정하는 숫자
파라미터를 알면 확률분포의 형태를 알 수 있다.

정규분포의 특성과 표준화의 개념
정규분포 
<정규분포 공식>
평균 u와 표준편차 시그마라는 2가지 파라미터로 확률분포가 정해진다.
u=0, 시그마=1인 정규분포 N(0,1)을 표준정규분포라 함.
정규분포 특징
1. 평균u를 중심으로 한 종형으로, 좌우대칭 분포
2. 평균u 근처에 값이 가장 많고, 평균u에서 멀어질수록 적어진다.
3. 키나 몸무게 등 정규분포로 근사할 수 있는 현상이 많다.

성질
1. u-sigma 부터 u+sigma까지 범위에 값이 있을 확률: 약 68%
2. u-2sigma 부터 u+2sigma까지 범위에 값이 있을 확률: 약 95%
3. u-3sigma 부터 u+3sigma까지 범위에 값이 있을 확률: 약 99.7%

표준화: 확률변수x 또는 데이터의 평균과 표준편차를 이용하여 다음과 같이 계산하면
z=(x-u)/sigma
평균0, 표준편차 1로 변환할 수 있는데 이를 표준화라고 하며 z값이라고 부르기도 함

평균과의 거리가 표준편차의 몇 배인가를 나타내기 때문에, 본래의 u나 sigma와 상관없이 분포 안에서 어디에 위치하는가를 알 수 있다.


## 06. 추론통계를 배우기 전에

```
✅ 학습 목표 :
* '데이터를 얻는다는 것'의 의미를 이해한다.
* 무작위추출의 개념과 통계적 필요성을 설명할 수 있다.
* 추론통계의 직관적 의미를 이해한다
```

데이터를 얻는다는 것: 모집단에 포함된 전체 값으로 구성된 분포에서 일부를 추출하는 것

무작위추출의 개념과 통계적 필요성
무작위추출: 데이터를 얻을 때 모집단에 포함된 요소를 하나씩 무작위로 선택하여 추출하는 방식
데이터의 실현값은 확률분포에서 무작위로 발생하도록 한 값이라고 생각해야 한다. 작은 값을 얻었다고 해서 표본으로 채택하지 않거나, 큰 값이라 해서 채택하는 등, 무작위가 아닌 방식을 사용해서는 안된다.

추론통계의 직관적 의미(된장국 맛을 보는 상황으로 이해)
1. 정말로 알고자 하는 것은 표본 데이터가 아니라 모집단이다.
2. 모집단의 모든 요소를 다 조사하는 전수조사는 어렵다.
3. 작은 크기의 표본으로도 모집단을 추론할 수 있다.
4. 표본을 추출할 때는 무작위로 추출해야 한다.


## 07. 표본오차와 신뢰구간

```
✅ 학습 목표 :
* 표본오차의 개념을 이해한다.
* 중심극한정리의 기본 개념과 역할을 이해한다.
* 신뢰구간의 의미와 활용 방법을 이해한다.
```

표본오차의 개념 이해
표본오차: '정말로 알고 싶은 것'과 '실제로 손 안에 있는 데이터' 사이에 생기는 어긋남(오차)
- 무작위로 표본을 고르는 데서 발생하는 피할 수 없는 오차이다.

중심극한정리의 기본 개념과 역할
중심극한정리: 모집단이 어떤 분포이든 간에, 표본크기 n이 커질수록 표본평균의 분포는 정규분포로 근사할 수 있다.
표본크기n이 커질수록 표본평균과 모집단평균 사이의 어긋남이 평균적으로 작아진다.

신뢰구간의 의미와 활용 방법 이해
신뢰구간은 표본에서 구한 모집단 u의 추정값을 어느 정도 신뢰할 수 있는지를 나타낸다고 할 수 있다.

00% 신뢰구간: "00%의 확률로 이 구간에 모집단평균 u가 있다"
u가 확률적으로 변화하여 그 구간에 포함되는 것이 아니라, 모집단에서 표본을 추출하여 00% 신뢰구간을 구하는 작업을 100번 반복했을 때 평균적으로 그 구간에 u가 포함되는 것이 00번이라는 뜻이다.

# 2️⃣ 확인 문제

## 문제 1.

> **🧚Q. 한 기업이 신제품 출시 후 “소비자 전반의 만족도”를 파악하고자 한다.
이를 위해 전체 소비자 중 일부를 무작위로 선택해 설문을 실시했고, 평균 만족도가 높게 나타났다.
그러나 내부 회의에서 다음과 같은 의견이 나왔다.

“설문 결과가 좋으니, 우리 제품은 소비자 전체에게도 만족도가 높다고 결론 내려도 되지 않을까?”
> **이 결론의 타당성을 판단하기 위해 통계학적으로 가장 적절한 설명은 무엇인가?**

~~~
1️⃣ 일부 소비자를 조사했으므로 이는 전수조사이며, 결과는 그대로 신뢰할 수 있다.
2️⃣ 표본의 평균이 높게 나타났다면, 모집단의 평균도 반드시 높다고 볼 수 있다.
3️⃣ 표본이 모집단을 적절히 대표한다면, 표본 결과를 통해 모집단의 특성을 추론할 수 있다.
4️⃣ 데이터 분석의 목적은 결과를 빠르게 도출하는 것이므로 표본의 대표성은 중요하지 않다.
~~~



<!--학습한 개념을 활용하여 자유롭게 설명해 보세요.-->

```
정답: 3️⃣, 우리기 모집단 전체의 데이터를 직접 분석하기는 어렵기 때문에 모집단의 일부인 표본을 추출하여 데이터를 분석하게 된다. 이때 신뢰구간이 높다면 표본이 모집단을 적절히 대표한다고 할 수 있다. 만약 표본이 모집단을 적절히 대표한다면, 표본 결과를 통해 모집단의 특성을 추론할 수 있으므로 정답은 3️⃣이다.
```
